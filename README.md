# 1.安装教程

1.下载releases文件，拷贝linux-do-oauth2.tar、 docker-compose.yml 、 start.sh 至同一文件夹

2.在当前目录下创建db目录，将data.db放入db目录中，举例：
```
- home
    - helper（此为当前目录）
        linux-do-oauth2.tar
        docker-compose.yml
        start.sh
        - db
            data.db
```

3.修改docker-compose.yml里的必填参数

4.执行命令：sh start.sh
